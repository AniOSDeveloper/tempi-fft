# TempiFFT


## Description

TempiFFT demonstrates how to input audio via AVFoundation for recording or processing and implements an FFT to display a real-time spectrum plot of incoming audio.



## Technologies

- Swift
- iOS
- AVFoundation
- DSP (Fast Fourier Transform)
- CoreGraphics
